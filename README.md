# AI Chatbot
AI Chatbot is a conversational application designed to interact with users in natural language. The project aims to provide intelligent, context-aware responses, making communication with the system simple and user-friendly.

## Features
- Natural language conversation
- User-friendly interface
- Real-time response generation
- Extensible architecture for adding new features
- Integration-ready with AI models and APIs

## Technologies Used
Languages:
python
HTML / CSS
JavaScript

Frameworks & Libraries:
Streamlit
Groq SDK
python-dotenv

AI:
Groq API
LLaMA 3.3 70B, LLaMA 3.1 8B, Mixtral 8x7B, Gemma2 9B

## Purpose
The goal of this project is to explore conversational AI and build a chatbot capable of assisting users with information, answering questions, and performing basic tasks.
